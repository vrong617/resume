GitHub Pages resume: https://vrong617.github.io

## Deploy to GitHub Pages

1. Create a new repo named `vrong617.github.io` at https://github.com/new
2. Run in this folder:
   ```bash
   git init
   git add .
   git commit -m "init: resume"
   git remote add origin git@github.com:vrong617/vrong617.github.io.git
   git push -u origin main
   ```
3. Go to repo **Settings → Pages → Branch: main / (root) → Save**

Site will be live at `https://vrong617.github.io` in ~1 minute.
